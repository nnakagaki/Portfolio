== My portfolio web app

= live at www.norrisnakagaki.com

to run locally
* download zip-file or run <tt>git clone https://github.com/nnakagaki/Portfolio.git</tt>
* <tt>bundle install</tt>
* <tt>bundle exec rake db:create db:migrate</tt>
* <tt>rails server</tt>
* go to localhost:3000 to view page

Asteroids and Snake are modified within this app to be the most updated version.
